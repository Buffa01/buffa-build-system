# Decision Making

Define como tomar decisiones tecnicas y de producto.

## Cuando usar ADR

- Cambio de arquitectura.
- Cambio de base de datos.
- Cambio de auth.
- Cambio de proveedor externo.
- Tradeoff de seguridad, costo o performance.

## Criterios

- Valor de negocio.
- Riesgo.
- Costo de reversibilidad.
- Complejidad.
- Operacion.
