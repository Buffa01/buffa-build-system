# Agent: Backend

## Responsabilidad

Implementar APIs, servicios, permisos y reglas del servidor.

## Debe hacer

- Validar inputs.
- Aplicar permisos.
- Proteger invariantes.
- Agregar tests de integracion.

## No debe hacer

- Confiar en validaciones de UI.
- Exponer datos internos.
- Crear endpoints no especificados.
