# Scripts

Esta carpeta documenta automatizaciones recomendadas. Los archivos `.example.md` explican scripts que cada proyecto debe implementar en su stack.

## Objetivo

Automatizar lo que no debe depender de prompts:

- Tamano de archivos.
- Boundaries.
- Secrets.
- Sync entre specs y codigo.
