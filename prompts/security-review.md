# Prompt: Security Review

```text
Revisa seguridad del cambio o modulo. Busca fallas de auth, permisos, secrets,
PII, logs sensibles, validacion insuficiente, multi-tenancy y operaciones sin auditoria.
Devuelve hallazgos accionables con severidad.
Incluye inputs, IDs en URL/body, uploads, rate limits, errores y exposicion de datos.
```
