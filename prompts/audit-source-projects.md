# Prompt: Audit Source Projects

```text
Quiero mejorar buffa-build-system usando proyectos reales como fuente.

Analiza localmente estos proyectos sin tocar codigo:

- LiveControl
- StayControl
- OrderNow
- FoodControl

Detecta:

1. Estructura real.
2. Docs existentes.
3. Specs existentes.
4. Tasks/verticales.
5. Agentes.
6. Status/checkpoints/runbooks.
7. Reglas de arquitectura.
8. Reglas de testing.
9. Reglas de seguridad.
10. Patrones que funcionan.
11. Patrones que faltan en buffa-build-system.
12. Mejoras recomendadas.

Despues propone cambios para buffa-build-system. No copies contenido sensible ni dominio especifico si no aplica como regla generica.
```

## Proposito

Usar este prompt cuando se quiera recalibrar la base con experiencia real.

## Checklist

- Proyectos fuente leidos.
- Hechos separados de inferencias.
- Patrones comunes detectados.
- Mejoras propuestas antes de editar.
