# Prompt: Create New Feature Spec

```text
Crea una spec para esta feature usando specs/modules/module-template.
Incluye PRD, reglas, flows, permisos, API, data model, test plan y decisiones abiertas.
Separa alcance actual de futuro. No inventes reglas; marca preguntas.

La spec minima debe responder:
que problema resuelve, para que usuario, que incluye, que excluye,
que reglas aplica, que datos usa, que permisos necesita, que errores puede tener,
como se prueba y que decisiones quedan abiertas.
```
