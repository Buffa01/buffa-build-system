# ADR

Architecture Decision Records guardan decisiones importantes.

## Reglas

- Una decision por ADR.
- Contexto claro.
- Opciones consideradas.
- Consecuencias explicitas.
- Estado actualizado.

## Estados

- Proposed
- Accepted
- Superseded
- Rejected
