# UI UX Standards

Define criterios de experiencia para productos digitales.

## Principios

- La UI debe apoyar el trabajo real del usuario.
- Priorizar flujos frecuentes y criticos.
- Mostrar estados de carga, error, exito y vacio.
- Evitar pantallas decorativas si el usuario necesita operar.
- Usar lenguaje del dominio.

## Reglas

- Cada accion destructiva requiere confirmacion o undo.
- Cada formulario debe validar antes de enviar.
- Cada lista critica debe tener busqueda, filtros o estados claros si aplica.
- Los errores deben explicar que paso y que puede hacer el usuario.

## Ejemplo

```text
En un POS, el boton de cobrar debe ser visible, rapido y resistente a errores.
```
